# Домашнє завдання

## Мета

Закріпити роботу з шаблонами та довести сторінку "Про себе" в проєкті до мінімально цілісного стану.

## Завдання

Створіть або доробіть/розширте сторінку персонального проєкту:
  - Створіть view detail(pk) і передайте у context: об'єкт, список пов'язаних елементів, дату
  - У шаблоні виведіть назву через |upper, опис через truncatewords:5, дату через |date:"d.m.Y"
  - Виведіть пов'язані елементи через {% for %} — якщо список порожній, покажіть "Нічого немає"
  - ⭐ Бонус: Створіть base.html з {% block content %} і успадкуйте detail.html від нього (будемо вивчати на наступному занятті)

## Що здати

- Код сторінок і шаблонів.
- Скріншоти або короткий опис того, що вже працює.

## На що звернути увагу

- HTML має бути охайним і читабельним.
- У шаблон не варто переносити зайву бізнес-логіку.
- Дані на сторінці повинні бути пов'язані з тематикою проєкту.
