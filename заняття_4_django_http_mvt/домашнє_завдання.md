# Домашнє завдання

## Мета

Закріпити маршрутизацію, базові `view` і розуміння MVT у власному Django-проєкті.

## Завдання

1. Реалізувати щонайменше три маршрути для свого проєкту.
2. Для одного маршруту використати параметр шляху, а для іншого опрацювати query string.
3. Додати один приклад переадресації або повернення нестандартного HTTP-статусу.

## Що здати

- Код Django-проєкту з готовими маршрутами і `view`.

## На що звернути увагу

- Маршрути мають бути логічними та читабельними.
- `View` повинні робити одну просту й зрозумілу дію.
- Не треба поки ускладнювати проєкт; головне, щоб цикл запит-відповідь став зрозумілим і робочим.
