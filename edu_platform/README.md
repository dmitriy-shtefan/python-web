# Education Platform

Навчальний Django-проєкт, який ми поступово розвиваємо під час курсу.

На поточному етапі проєкт працює без власних моделей та без налаштування бази даних: дані для сторінок поки що зберігаються у Python-списках у views.

## Як запустити проєкт локально

### 1. Отримайте копію (клон) репозиторію

```bash
git clone https://github.com/dmitriy-shtefan/python-web.git
cd python-web/edu_platform
```

### Якщо репозиторій вже є

Якщо ви вже раніше виконували `git clone`, повторно клонувати репозиторій не потрібно. Перейдіть у папку проєкту та отримайте останні зміни:

```bash
cd python-web
git pull
cd edu_platform
```

Після цього продовжуйте з наступного кроку.

### 2. Перевірте чи django встановлений

```bash
django-admin --version
```

### 3. Встановіть Django, якщо потрібно

```bash
pip install django
```

### 4. Запустіть development-сервер

```bash
python manage.py runserver
```

Після запуску відкрийте в браузері:

```text
http://127.0.0.1:8000/courses/
```

## Важливо на цьому етапі

- Базу даних, моделі та Django Admin будемо підключати пізніше.
- Якщо порт `8000` зайнятий, можна запустити сервер на іншому порту:

```bash
python manage.py runserver 8001
```
