# Домашнє завдання

## Мета

Зібрати базовий макет персонального проєкту та впорядкувати шаблонну структуру.

## Завдання

1. Створити базовий шаблон і винести в нього спільні частини сторінок.
2. Підключити мінімум один CSS-файл і, за потреби, зображення або JS.
3. Перевірити, що всі основні сторінки користуються спільним макетом і коректно підключають static files.

## Challenge

Виконати 1-2 додаткові пункти:

- винести картку елемента в `includes/item_card.html` і використати її в циклі;
- додати блок `{% block extra_css %}` або `{% block extra_js %}`;
- підключити простий `main.js` через `{% static %}`;
- додати зображення з `static/app_name/images/`;

## Що здати

- Оновлений код шаблонів.
- Короткий список сторінок, які вже користуються базовим макетом.
- Короткий список створених або змінених файлів.
